Se debe implementar un modelo de mixtura de gausianas mediante el algoritmo de esperanza-maximizacin (EM). Este modelo se utiliza para estimar una densidad de probabilidad a partir de $N$ puntos en $D$ dimensiones, $\mathcal{D}=\{\mathbf{x}_i\}_{i=1}^N$, basado en $K$ gausianas

$$ p(\mathbf{X}) = \sum_{k=1}^K \pi_k \mathcal{N} (\mathbf{X}; \mathbf{\mu_k}, \Sigma_k) $$

donde $\pi_k$ es la probabilidad de observar un punto en la gausiana $k$ con $\sum_{k=1}^K \pi_k = 1$ y donde $\mathbf{\mu_k}, \Sigma_k$ son la media y la matriz de covarianza respectivamente de la gausiana $k$.

Para resolver este problema vamos a plantearlo con la siguiente red de bayes:

donde la variable $\mathbf{Z}$ indica la gausiana a la que pertenece cada dato codificado como 1-en-K, esto es, $\mathbf{Z}$ es un vector binario de K componentes donde una y solo una vale 1 y el resto 0. Si $z_{ik}=1$ entonces el ejemplo $\mathbf{X}_i$ pertence a la gausiana $k$. Donde

$$ p(\mathbf{X}) = \sum_{k=1}^K p(z_k=1) p(\mathbf{X}|z_k=1) $$

donde $p(z_k=1)=\pi_k$ y $p(\mathbf{X}|z_k=1) = \mathcal{N} (\mathbf{X}; \mathbf{\mu_k}, \Sigma_k) $.

Para aplicar el algoritmo EM tenemos que la variable no observada es la $\mathbf{Z}$ para todos los ejemplos, por lo que en el paso E debemos estimar la probabilidad de observar 1 para cada uno de las $K$ entradas en $Z$ para cada uno de los ejemplos en $\mathcal{D}$.

En el paso E, estimaremos $D \times K$ valores de la siguiente forma:

Paso E:

$$ q_i(z_k=1) = \frac{p(z_k=1)p(\mathbf{X}|z_k=1)}{\sum_{k=1}^K p(z_k=1)p(\mathbf{X}|z_k=1)} $$

En el paso M lo que debemos hacer es estimar los parmetros del modelo a partir de los datos como se indica a continuacin.

Paso M:

$$ \pi_k = \frac{1}{N} \sum_{i=1}^N q_i(z_k=1) $$$$ \mathbf{\mu}_k = \frac{1}{\sum_{i=1}^N q_i(z_k=1)} \sum_{i=1}^N q_i(z_k=1) \mathbf{x}_i $$$$ \Sigma_k = \frac{1}{\sum_{i=1}^N q_i(z_k=1)} \sum_{i=1}^N q_i(z_k=1) (\mathbf{x}_i - \mathbf{\mu}_k)(\mathbf{x}_i - \mathbf{\mu}_k)^T $$

Finalmente la log-verosimilitud de los datos la podemos escribir como

$$ ln p(\mathbf{X}|\pi, \mathbf{\mu}, \Sigma) = \sum_{i=1}^N ln \left[ \sum_{k=1}^K \pi_k \mathcal{N} (\mathbf{x}_i; \mathbf{\mu_k}, \Sigma_k) \right]$$

Problemas numéricos¶

En la implementacin de este mtodo surgen diversos problemas numricos:

  • La matriz de covarianza debe ser semi-definida positiva. Para garantizar que las estimaciones que tenemos de la matriz de covarianza lo cumplen, se puede aadir un valor pequeo de regularizacin a la diagonal de la matriz (~1e10-5).
  • El clculo de los valores de $q$ supone el clculo de una gausiana y puede darse el caso de que para algn ejemplo y para todo $k$ el valor sea 0. Para evitar esto se puede sumar una cantidad muy pequea siempre a todos los valores de $q$ antes de normalizar. Adicionalmente se recomienda calcular esta probailidad como log prob y luego utilizar la funcin np.logsumexp para normalizar en el espacio de logprobs.
  • Finalmente, para evitar problemas numricos al invertir la matriz de covarianza, se recomienda usar la descomposicin de Choleski esto es: covinv = scipy.linalg.cho_solve(scipy.linalg.cho_factor(covariaza), I), donde I es la matriz identidad.

import matplotlib.transforms as transforms


def make_ellipse(cov, mean, ax, n_std=3.0, **kwargs):
    """
    Adaptado de: https://matplotlib.org/stable/gallery/statistics/confidence_ellipse.html

    En kwargs se pueden poner todo tipo de argumentos para dibujar la elipse. Por ejemplo
    para que la elipse sea naranja semitransparente con borde rojo sera:

        def make_ellipse(cov, mean, ax, n_std=3.0, facecolor='orange', edgecolor='red', alpha=0.5)

    """

    pearson = cov[0, 1] / np.sqrt(cov[0, 0] * cov[1, 1])
    # Using a special case to obtain the eigenvalues of this
    # two-dimensionl dataset.
    ell_radius_x = np.sqrt(1 + pearson)
    ell_radius_y = np.sqrt(1 - pearson)
    ellipse = mpl.patches.Ellipse(
        (0, 0), width=ell_radius_x * 2, height=ell_radius_y * 2, **kwargs
    )

    # Calculating the stdandard deviation of x from
    # the squareroot of the variance and multiplying
    # with the given number of standard deviations.
    scale_x = np.sqrt(cov[0, 0]) * n_std
    mean_x = mean[0]

    # Calculating the stdandard deviation of y ...
    scale_y = np.sqrt(cov[1, 1]) * n_std
    mean_y = mean[1]

    transf = (
        transforms.Affine2D()
        .rotate_deg(45)
        .scale(scale_x, scale_y)
        .translate(mean_x, mean_y)
    )

    ellipse.set_transform(transf + ax.transData)

    color = kwargs["edgecolor"] if "edgecolor" in kwargs else "red"

    ax.scatter(mean_x, mean_y, marker="x", c=color)

    return ax.add_patch(ellipse)

def generate_data(N=100, K=3, rotate=True):
    X, y = make_blobs(N, centers=K)

    rotate = True
    transformation = np.array([[1, 0], [0, 1]])

    transformations = []
    for y_k in np.unique(y):
        mask = y == y_k
        if rotate:
            transformation = np.array(
                [
                    [(0.2 + np.random.rand()) * 2, 0.5 - np.random.rand()],
                    [0.5 - np.random.rand(), (0.2 + np.random.rand()) * 2],
                ]
            )

        X[mask, :] = np.dot(X[mask, :], transformation)
        transformations.append(transformation)

    return X, y, transformations

colors = ["royalblue", "mediumseagreen", "crimson"]

def plot_data(X, y, transformations=None, means=None, covs=None, title=None):
    fig = plt.figure(figsize=(6, 6))
    for y_k in np.unique(y):
        mask = y == y_k
        plt.scatter(X[mask, 0], X[mask, 1], label=y_k, color=colors[y_k])

        if transformations is not None:
            make_ellipse(
                transformations[y_k],
                np.mean(X[mask, :], axis=0),
                ax=fig.axes[0],
                facecolor="none",
                edgecolor="indigo",
            )

        elif means is not None and covs is not None:
            mu = means[y_k, :]
            cov = covs[y_k, :]
            print("======== Class {0} ========".format(y_k))
            print("mu: {0}".format(mu))
            print("cov: {0}".format(cov))
            make_ellipse(cov, mu, fig.axes[0], facecolor="none", edgecolor="indigo")

    fig.axes[0].axis("equal")
    _ = plt.legend()
    if title:
        plt.title(title)
    plt.show()

En primer lugar, vamos a simplificar las operaciones que involucran el clculo de la log-verosimilitud:

$$ \log p(\mathbf{X}|\pi, \mathbf{\mu}, \Sigma) = \log \left[ \sum_{k=1}^K \exp\log{\pi_k} + \log{ \mathcal{N} (\mathbf{X}; \mathbf{\mu_k}, \Sigma_k)} \right]$$

utilizando la descomposicin de Cholesky de la matriz de precisin para el trmino $\log{ \mathcal{N} (\mathbf{X}; \mathbf{\mu_k}, \Sigma_k)}$.

Sea $$\log{ \mathcal{N} (\mathbf{X}; \mathbf{\mu_k}, \Sigma_k)} = -\frac{1}{2}k\log 2\pi - \frac{1}{2}\log|\Sigma| - \frac{1}{2}(x-\mu)^T\Sigma^{-1}(x-\mu)$$

La matriz de precisin es la inversa de la matriz de covarianza

$$\Lambda = \Sigma^{-1}$$

Utilizando la descomposicin de Cholesky

$$\Lambda = L L ^T$$

El trmino que involucra el la distancia de Mahalanobis se simplifica a

$$ - \frac{1}{2}(x-\mu)^T\Sigma^{-1}(x-\mu) = - \frac{1}{2}(x-\mu)^T\Lambda(x-\mu) = - \frac{1}{2}(x-\mu)^TL L^T(x-\mu) = - \frac{1}{2} || L^T(x-\mu) ||^2$$

El trmino que involucra el determinante se simplifica a $$ - \frac{1}{2}\log|\Sigma| = - \frac{1}{2} \log |\Lambda^{-1}| = \frac{1}{2} \log |\Lambda| = \frac{1}{2} \log |L L^T| = \log |L|$$

Por tanto, la expresin final correspondiente a la gaussiana es:

$$\log{ \mathcal{N} (\mathbf{X}; \mathbf{\mu_k}, \Sigma_k)} = -\frac{1}{2} (k\log 2\pi + || L^T(x-\mu) ||^2) + \log|L|$$

from sklearn.cluster import KMeans
import warnings


class GMM:
    """
    Sugerencia para la organizacin del cdigo
    """

    def __init__(
        self,
        n_components: int = 3,
        n_iterations: int = 100,
        tol: float = 1e-3,
        reg_covar: float = 1e-5,
        init_params: str = "kmeans",
        seed: int = None,
    ):
        self.n_components = n_components
        self.n_iterations = n_iterations
        self.tol = tol
        self.reg_covar = reg_covar
        self.init_params = init_params
        self.seed = seed

    def _estimate_gaussian_parameters(self, X, resp):
        """
        Estimate the Gaussian distribution parameters.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)
            The input data array.
        resp : array-like of shape (n_samples, n_components)
            The responsibilities for each data sample in X.

        Returns
        -------
        priors : array-like of shape (n_components,)
            The priors of data samples in the current components.
        means : array-like of shape (n_components, n_features)
            The centers of the current components.
        covs : array-like of shape (n_components, n_features, n_features)
            The covariance matrix of the current components.
        """

        # Get number of samples and features from input data shape
        n_samples, n_features = X.shape

        # Compute priors, probability of each component
        priors = np.sum(resp, axis=0)

        # Compute mean
        means = resp.T @ X / priors[:, None]

        # ... and init covariances matrix
        covs = np.empty((self.n_components, n_features, n_features))

        # For every K entries
        for k in range(self.n_components):
            # Get mean
            X_centered = X - means[k]

            # Compute covariances
            covs[k] = resp[:, k] * X_centered.T @ X_centered / priors[k]

            # Add regularization term at the diagonal
            covs[k].flat[:: n_features + 1] += self.reg_covar

        # Divide by N all weights to get prior
        priors /= n_samples

        return priors, means, covs

    def _compute_precision_cholesky(self):
        """
        Compute the Cholesky decomposition of the precision matrix, the inverse of covariance matrix.

        Returns
        -------
        precisions_cholesky : array-like of shape (n_components, n_features, n_features)
            The cholesky decomposition of sample precisions of the current
            components.
        """

        # Initiate array
        n_features = self.covs_.shape[1]
        precisions_cholesky = np.empty((self.n_components, n_features, n_features))

        for k in range(self.n_components):
            # Compute Cholesky decomposition from SciPy
            cov_chol = np.linalg.cholesky(self.covs_[k])

            # Compute precision matrix from inverse of covariance matrix
            precisions_cholesky[k] = scipy.linalg.solve_triangular(
                cov_chol, np.eye(n_features), lower=True
            ).T

        return precisions_cholesky

    def _initialize(self, X):
        """
        Initialization of the Gaussian mixture parameters.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)

        """

        n_samples, _ = X.shape

        # Cluster with kmeans prediction
        if self.init_params == "kmeans":
            # Init Z
            resp = np.zeros((n_samples, self.n_components))

            # Get predicted labels
            labels = (
                KMeans(n_clusters=self.n_components, n_init=1, random_state=345)
                .fit(X)
                .labels_
            )

            # Fill with 1 if X_i belongs to gaussian K
            resp[np.arange(n_samples), labels] = 1

        # Cluster with random initialitation
        elif self.init_params == "random":
            # Compute random values
            resp = np.random.default_rng(seed=self.seed).random(
                (n_samples, self.n_components)
            )

            # Probabilities normalization
            resp /= resp.sum(axis=1)[:, None]

        else:
            raise ValueError(
                f"Please, select 'kmeans' or 'random' init_param as initialization method."
            )

        # Compute prior weights, means and covariances
        self.priors_, self.means_, self.covs_ = self._estimate_gaussian_parameters(
            X, resp
        )

        # ... and precompute the Cholesky decomposition of covariance matrixes
        self.precisions_cholesky_ = self._compute_precision_cholesky()

    def _estimate_log_gaussian_prob(self, X):
        """
        Estimate the log Gaussian probability.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)

        Returns
        -------
        log_prob : array, shape (n_samples, n_components)
        """

        n_samples, n_features = X.shape

        log_prob = np.empty((n_samples, self.n_components))

        # Compute Mahalanobis term
        for k in range(self.n_components):
            log_prob[:, k] = np.sum(
                np.square((X - self.means_[k]) @ self.precisions_cholesky_[k]), axis=1
            )

        #  Compute the determinant of the precision matrix for each component
        log_det = np.log(np.linalg.det(self.precisions_cholesky_))

        # Compute log gaussian probability
        return -0.5 * (n_features * np.log(2 * np.pi) + log_prob) + log_det

    def _estimate_log_prob(self, X):
        """
        Estimate the log-probabilities, log P(X | Z) + log priors.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)

        Returns
        -------
        log_prob : array, shape (n_samples, n_component)
        """

        # Compute priors
        log_priors = np.log(self.priors_)

        # Compute log probabilities for each sample
        log_gaussian_prob = self._estimate_log_gaussian_prob(X)

        return log_priors + log_gaussian_prob

    def _e_step(self, X):
        """
        E step.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)

        Returns
        -------
        log_prob_norm : float
            Mean of the logarithms of the probabilities of each sample in X
        log_responsibility : array, shape (n_samples, n_components)
            Logarithm of the posterior probabilities (or responsibilities) of
            the point of each sample in X.
        """

        # Estimate the log probabilities
        log_prob = self._estimate_log_prob(X)

        # Normalize log-probs space
        log_prob_norm = scipy.special.logsumexp(log_prob, axis=1)

        # Posterior probabilities
        log_resp = log_prob - log_prob_norm[:, None]

        return np.mean(log_prob_norm), log_resp

    def _m_step(self, X, log_resp):
        """
        M step.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)

        log_resp : array-like of shape (n_samples, n_components)
            Logarithm of the posterior probabilities (or responsibilities) of
            the point of each sample in X.
        """

        # Estimate model parameters from given data
        self.priors_, self.means_, self.covs_ = self._estimate_gaussian_parameters(
            X, np.exp(log_resp)
        )

        # Compute Cholesky decomposition for covariance matrix
        self.precisions_cholesky_ = self._compute_precision_cholesky()

    def fit(self, X):
        """
        Estimate model parameters with the EM algorithm.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)
            List of n_features-dimensional data points. Each row
            corresponds to a single data point.

        Returns
        -------
        self : object
            The fitted mixture.
        """

        # Params init
        converged = False
        log_prob = -np.infty
        self.log_probs = []

        # Initiate model
        self._initialize(X)

        for it in range(1, self.n_iterations):
            # E-step
            current_log_prob, log_resp = self._e_step(X)
            self.log_probs.append(current_log_prob)

            # M-step
            self._m_step(X, log_resp)

            # Check convergence criteria
            if abs(current_log_prob - log_prob) < self.tol:
                converged = True
                break

            # Save next log prob
            log_prob = current_log_prob

        # Convergence warning
        if converged == False:
            warnings.warn("EM method did not converge.")

        self.total_iter = it

        return self

    def predict(self, X):
        """
        Predict the labels for the data samples in X using trained model.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)
            List of n_features-dimensional data points. Each row
            corresponds to a single data point.

        Returns
        -------
        labels : array, shape (n_samples,)
            Component labels.
        """

        _, log_resp = self._e_step(X)
        return log_resp.argmax(axis=1)

    def fit_predict(self, X):
        """
        Estimate model parameters using X and predict the labels for X.

        Parameters
        ----------
        X : array-like of shape (n_samples, n_features)
            List of n_features-dimensional data points. Each row
            corresponds to a single data point.

        Returns
        -------
        labels : array, shape (n_samples,)
            Component labels.
        """

        return self.fit(X).predict(X)
