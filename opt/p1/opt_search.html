def plot_solution_2d(func, real_minimum, computed_minimum, a=-3, b=5):
    x = np.linspace(a, b, 1000)
    plt.plot(x, func(x))
    plt.title(r"$f1(x)$")

cuyo mnimo sabemos que se alcanza en $f_1(x^*)=-1$ para $x^*=0$.

def plot_solution_3d(func, real_minimum, computed_minimum, a = -100, b = 100):
    fig = plt.figure()
    ax = plt.axes(projection='3d')

    # Data for a three-dimensional line
    xline = np.linspace(a, b, 1000)
    yline = np.linspace(a, b, 1000)
    x = np.array(list(zip(xline,yline)))
    zline = list(map(func, x))

    ax.plot3D(xline, yline, zline)
    plt.title(r"$f2(x)$")
    ax.scatter3D(real_minimum[0], real_minimum[1], func(real_minimum), color="red", marker="*", label="Real minimum")
    ax.scatter3D(computed_minimum[0], computed_minimum[1], func(computed_minimum),  color="green", s = 80, facecolor = 'none', marker="o", label="Computed minimum")
    plt.legend(loc = 'best')
    plt.show()

Método de Hooke y Jevees¶

Se muestra, a travs de un ejemplo, el mtodo de optimizacin de Hooke y Jevees implementado como HJ_search en search_methods.py.

Para la bsqueda del $\lambda$ mnimo en los pasos 1 y 2 del algoritmo visto en clase, se utiliza una variante del mtodo que utiliza dos parmetro complementarios: $\beta$ y $\alpha$ que miden la tasa de reduccin y el factor de aceleracin del algorito respectivamente.

Método Newton (multivariante)¶

Se prueba el mtodo de optimizacin de Newton multivariante implementado como newton_search en search_methods.py, utilizando de nuevo la funcin $f_2$ a minimizar.

Para ello, es necesario calcular el Hessiano de $f_2$:

$$\nabla {f_2}(x) = (4(x_1-2)^3+2(x_1-2x_2), -4(x_1-2x_2))$$$$H f_2(x) = \begin{pmatrix} 12(x-2)^2+2 & -4\\ -4 & 8 \end{pmatrix}$$
In [12]:
def grad_f2(x):
    return np.array([4*(x[0]-2)**3 + 2*(x[0]-2*x[1]), -4*(x[0]-2*x[1])], dtype=np.float64)
    

def H_f2(x):
    return np.matrix([[12*(x[0]-2)**2 + 2, -4], [-4, 8]], dtype=np.float64)
